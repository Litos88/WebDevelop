Comandos Git mas utilizados:

  1. git clone - Clonamos un proyecto en un directorio local
  2. git status - Vemos el estado del proyecto, branches, commits pendientes
  3. git add archivo - Preparamos un archivo para ser guardado"
  4. git commit -m/am "message - Guardamos una versión"
  5. git push - Subimos el archivo guardado a GitHub
  6. git pull - Bajamos al directorio local la versión del archivo en Github
  7. git branch - Muestra las ramas del proyecto
  8. git checkout -b branch_name - Creamos una nueva rama
  9. git checkout branch_name - Nos desplazamos a una rama (Se deben hacer commit de los cambios realizados o stash)
  10. git merge branch_name - Guardamos una versión"
